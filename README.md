# smart-waste-management-system

This project is an AI-based Smart Waste Management and Citizen Complaint System that allows users to report waste issues and helps authorities manage and resolve them efficiently.

---

##  System Overview

The system consists of three main components:

- Android Application (User side)
- Spring Boot Backend (API & database handling)
- Web Dashboard (Admin side)

---

##  Features

- Users can report waste issues with images
- Location-based complaint submission
- Real-time complaint tracking
- Admin dashboard to view and update complaints
- Efficient waste management workflow

##  Working Flow

1. User captures waste image using mobile app  
2. Complaint is sent to backend via REST API  
3. Backend processes and stores data in database  
4. Admin views complaints in web dashboard  
5. Admin updates complaint status
